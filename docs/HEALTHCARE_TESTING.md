# Healthcare-Specific Testing Checklist

## Medical Form Scenarios
- [ ] Patient intake forms
- [ ] Insurance enrollment forms
- [ ] Pre-appointment questionnaires
- [ ] Medical history forms
- [ ] Consent forms
- [ ] Forms with checkboxes/radio buttons

## Prescription Scenarios
- [ ] Clear, readable prescriptions
- [ ] Handwritten prescriptions
- [ ] Multiple medications on one image
- [ ] Prescriptions in different languages
- [ ] Blurry/low-quality images
- [ ] Prescriptions with stickers/labels

## Drug Interaction Testing
- [ ] Known dangerous combinations detected
- [ ] False positives minimized
- [ ] Allergy detection works
- [ ] Severity levels correct
- [ ] Recommendations appropriate
- [ ] Multiple prescriptions handled

## Diet Portal Testing
- [ ] Common conditions (diabetes, hypertension)
- [ ] Rare conditions
- [ ] Multiple conditions
- [ ] Medication-food interactions
- [ ] Meal plan generation
- [ ] Food compatibility checking

## Data Accuracy
- [ ] Medication names extracted correctly
- [ ] Dosages accurate
- [ ] Dates parsed correctly
- [ ] Insurance info extracted
- [ ] Medical terms understood
- [ ] Units of measurement correct

## Safety Critical
- [ ] No false negatives (missed interactions)
- [ ] Warnings are clear and actionable
- [ ] Medical advice disclaimers present
- [ ] "Consult doctor" messages shown
- [ ] No dangerous recommendations
- [ ] Error messages don't mislead

## Compliance
- [ ] HIPAA disclaimers visible
- [ ] Privacy policy accessible
- [ ] Terms of service shown
- [ ] Data handling disclosed
- [ ] User consent mechanisms
- [ ] Audit trails maintained
